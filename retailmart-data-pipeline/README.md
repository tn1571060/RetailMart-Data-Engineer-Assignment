# RetailMart Data Engineer Assignment

## Overview

This project implements a complete ETL (Extract, Transform, Load) pipeline for RetailMart sales data using Python, Pandas, NumPy, and SQLite.

The pipeline performs data ingestion, cleaning, transformation, SQL analysis, and reporting to generate meaningful business insights.

---

## Technologies Used

- Python 3
- Pandas
- NumPy
- SQLite3

---

## Features

- Read data from multiple CSV files
- Detect missing values
- Remove duplicate records
- Handle missing data
- Convert data types
- Merge datasets
- Calculate total revenue
- Perform revenue analysis
- Load processed data into SQLite
- Execute SQL queries
- Generate summary report
- Basic error handling

---

## SQL Queries Implemented

### Top 3 Best-Selling Products

Returns the three products with the highest total quantity sold.

### Revenue Per Store Per Day

Calculates daily revenue generated by each store.

---

## Files Included

### Input Files

- `sales_data.csv`
- `products.csv`
- `stores.csv`

### Generated Output Files

- `cleaned_retail_sales.csv` – Cleaned and transformed dataset generated by the pipeline.
- `retailmart.db` – SQLite database containing the processed retail sales table.

These output files are included so reviewers can directly inspect the generated results without running the pipeline. They can also rerun the pipeline to regenerate these files.

---

## How to Run

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Run the pipeline

```bash
python pipeline.py
```

The pipeline will automatically:

- Read the input CSV files
- Clean and transform the data
- Create `cleaned_retail_sales.csv`
- Create `retailmart.db`
- Execute SQL queries
- Display business insights and summary statistics
