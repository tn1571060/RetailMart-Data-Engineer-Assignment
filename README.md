# RetailMart Data Engineer Assignment

## Overview

This project is a simple ETL (Extract, Transform, Load) pipeline developed as part of a Data Engineer Technical Assignment.

The pipeline reads retail sales data from multiple CSV files, performs data cleaning and transformation using Pandas and NumPy, loads the processed data into a SQLite database, executes SQL queries for business insights, and generates a summary report.

---

## Technologies Used

- Python 3
- Pandas
- NumPy
- SQLite3

---

## Project Structure

```
RetailMart_Data_Engineer_Assignment/
│
├── pipeline.py
├── sales_data.csv
├── products.csv
├── stores.csv
├── README.md
└── requirements.txt
```

---

## Features

- Load data from multiple CSV files
- Identify missing values
- Remove duplicate records
- Handle missing data
- Convert data types
- Merge multiple datasets
- Calculate total revenue
- Analyze revenue statistics
- Generate revenue by city report
- Load cleaned data into SQLite
- Execute SQL queries for business insights
- Generate summary report
- Basic error handling using try-except

---

## SQL Queries Implemented

### 1. Top 3 Best-Selling Products

Finds the top three products based on total quantity sold.

### 2. Revenue Per Store Per Day

Calculates total revenue generated by each store on each day.

---

## Output Generated

After running the pipeline, the following outputs are produced:

- Cleaned and merged dataset (`cleaned_retail_sales.csv`)
- SQLite database (`retailmart.db`)
- Revenue statistics
- Revenue by city
- Top 3 best-selling products
- Revenue per store per day
- Summary report

---

## How to Run

1. Clone the repository.

2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Make sure the following files are in the project folder:

- sales_data.csv
- products.csv
- stores.csv

4. Run the pipeline:

```bash
python pipeline.py
```
